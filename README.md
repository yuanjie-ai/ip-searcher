## ip-searcher 是什么

ip-searcher - 是一个离线IP地址定位库和IP定位数据管理框架，10微秒级别的查询效率，提供了众多主流编程语言的 `xdb`
数据生成和查询客户端实现。

## Install

```bash
pip install -U ip-searcher
```

## Usage

```python
from ip_searcher import search

print(search("117.136.66.124"))
```
